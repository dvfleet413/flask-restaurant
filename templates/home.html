Flask Restaurant

Welcome to Flask Restaurant!

This app is divided into two sections...and admin panel and an API.

Admin Panel

The admin panel is built with Flask and Jinja Templates. The purpose of this section of the app is to provide a user-friendly interface to interact with the app's database. Users can easily add or edit entrees using the admin panel. These entrees are then available through the API to other applications

API

The API provides access to the database for other applications. Available endpoints are:

  • "GET /api/entrees" - returns all entrees in the database
  • "GET /api/entrees/{id}" - returns the entree with matching id
